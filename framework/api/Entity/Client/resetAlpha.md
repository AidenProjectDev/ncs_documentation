# resetAlpha

```lua
NCs.Ped:resetAlpha(entityId)
```

### Arguments
| Argument | Type   | Optional | Default | Explanation           |
|----------|--------|----------|---------|-----------------------|
| entityId | number | false    | -       | The ID of the entity. |

### Examples
```lua
local client = PlayerPedId()
NCs.Ped.resetAlpha(client)
```
