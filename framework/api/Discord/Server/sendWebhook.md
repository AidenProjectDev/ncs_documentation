# sendWebhook

```lua
NCs.Discord:sendWebhook(webhook, details)
```

### Arguments
| Argument | Type   | Optional | Default | Explanation         |
|----------|--------|----------|---------|---------------------|
| webhook  | string | false    | -       | Link of the webhook |
| details  | string | false    | -       | Embed details.      |
