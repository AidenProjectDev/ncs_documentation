# showNotification

```lua
NCs.Player:showNotification(text, colorBg)
``` 

### Arguments
| Argument | Type   | Optional   | Default | Explanation                 |
|----------|--------|------------|---------|-----------------------------|
| text     | string | false      | -       | The text to display         |
| colorBg  | number | false      | -       | The color of the background |

### Examples
```lua
NCs.Player:showNotification("text")
```
